# 77. Combinations

## Medium

***

Given two integers `n` and `k`, return _all possible combinations of_ `k` _numbers out of the range_ `[1, n]`.

You may return the answer in **any order**.

&#x20;

**Example 1:**

<pre><code>Input: n = 4, k = 2
<strong>Output:
</strong>[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]</code></pre>

**Example 2:**

<pre><code>Input: n = 1, k = 1
<strong>Output:
</strong> [[1]]</code></pre>

&#x20;

**Constraints:**

* `1 <= n <= 20`
* `1 <= k <= n`
