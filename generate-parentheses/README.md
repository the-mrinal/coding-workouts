# Generate Parentheses

***

Given `n` pairs of parentheses, write a function to _generate all combinations of well-formed parentheses_.

&#x20;

**Example 1:**

<pre><code>Input: n = 3
<strong>Output:
</strong> ["((()))","(()())","(())()","()(())","()()()"]</code></pre>

**Example 2:**

<pre><code>Input: n = 1
<strong>Output:
</strong> ["()"]</code></pre>

&#x20;

**Constraints:**

* `1 <= n <= 8`
