# 633. Sum of Square Numbers

## Medium

***

Given a non-negative integer `c`, decide whether there're two integers `a` and `b` such that `a2 + b2 = c`.

&#x20;

**Example 1:**

<pre><code>Input: c = 5
<strong>Output:
</strong> true
<strong>Explanation:
</strong> 1 * 1 + 2 * 2 = 5</code></pre>

**Example 2:**

<pre><code>Input: c = 3
<strong>Output:
</strong> false</code></pre>

&#x20;

**Constraints:**

* `0 <= c <= 231 - 1`
