# Sort an Array

***

Given an array of integers `nums`, sort the array in ascending order.

&#x20;

**Example 1:**

<pre><code>Input: nums = [5,2,3,1]
<strong>Output:
</strong> [1,2,3,5]</code></pre>

**Example 2:**

<pre><code>Input: nums = [5,1,1,2,0,0]
<strong>Output:
</strong> [0,0,1,1,2,5]</code></pre>

&#x20;

**Constraints:**

* `1 <= nums.length <= 5 * 104`
* `-5 * 104 <= nums[i] <= 5 * 104`
